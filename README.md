# Interaction-Service

Микросервис для управления социальными взаимодействиями: лайками и закладками.

## Основные возможности
- **Лайки**: Можно лайкать посты, комментарии или сообщения.
- **Закладки**: Сохранение важного контента в личный список.
- **Счетчики**: Быстрое получение количества лайков для любого объекта.

---

## API Эндпоинты (Защищенные Гейтвеем)

### Лайки
- **POST** `/api/likes` — Поставить лайк.
- **DELETE** `/api/likes` — Убрать лайк.
- **GET** `/api/likes/count` — Получить количество лайков (параметры: `target_type`, `target_id`).

### Закладки
- **POST** `/api/bookmarks` — Добавить в закладки.
- **DELETE** `/api/bookmarks` — Удалить из закладок.
- **GET** `/api/bookmarks` — Список всех закладок пользователя.

---

## Технологии
- **Go 1.25** + **Gin**
- **PostgreSQL** (GORM)
- **Shared Auth Middleware**: Использует заголовки от API Gateway.
